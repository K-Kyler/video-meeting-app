# Using:
<br />• NodeJS 14.15.0
<br />• ExpressJS 4.17.1
<br />• PeerJS 0.6.1
<br />• SocketIO 3.1.1
<br />• MongoDB 4.4.4
<br />• PugEngine 3.0.0

# Run Project Step-by-Step:
<br />1/ Clone project: 
```
git clone https://github.com/K-Kyler/video-meeting-app.git
```
<br />2/ Import node_modules:
```
cd video-meeting-app
npm install
```
<br />3/ Setup database by MongoDB with name "webrtc"

<br />4/ Run:
```
npm start
```
